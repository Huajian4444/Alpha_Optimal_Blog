# 生成Iso代码

> 当模拟确认无误后,生成Iso文件到指定路径.让机器执行Iso文件开始加工

## 加工模式
| 图标 | 描述       |
|------|------------|
| ![播放](../../image/left.png)    | 输出左台面程序       |
| ![停止](../../image/right.png)    | 输出右台面程序       |
| ![前进](../../image/tandem.png) | 输出左台面程序.调用后处理文件的串联加工模式   |
| ![后退](../../image/alternately.png)| 输出左&右台面的程序.调用后处理文件的交替加工模式   |

> 注意:在生成Iso代码之前,必须选择需要的加工模式.它和模拟之类的是没有关系的。

## 生成Iso文件
- 文件格式由 后处理文件定义
- 输出路径由 后处理文件定义
- 文件的前缀和后缀由 后处理文件定义
