# 用户坐标 值输入(修改)

要详细了解AlphaOptimal的坐标系系统 和 用户坐标系 的概念和用法.详见 [坐标系](../other/CoordinateSystem.md)

## 编辑
1. 操作面板 -> 机器节点 -> 工件坐标系 -> 点击:**用户坐标系**.在下方会显示用户坐标系的属性控件
2. 修改 X Y Z的值,确认后,点击**应用**
3. 所有引用该 用户坐标系 的程序段会自动计算</p></p>
