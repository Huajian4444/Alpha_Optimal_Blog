# AlphaOptimal_v3 目录结构

<span style="color:blue;">本软件目录结构简洁明了，便于用户快速定位和理解各类文件及其作用</span>

```plaintext
AlphaOptimal_v3/
├── bin/
│   ├── automaticFixtures        # 存储用户夹具配置文件相关资料
│   ├── ConfigEditorLanguage     # 存储配置文件编辑器用的翻译文件
│   ├── language                 # AlphaOptimal存储软件翻译文件
│   ├── machineSample            # 存储机床配置文件
│   ├── UserIdGenerator          # 软件安装时必要的注册工具
│   ├── UserPostProcessor        # 存储用户后处理文件
│   ├── data.ini                 # 存储软件相关的自定义信息
│   ├── log.txt                  # 日志文件
│   ├── toollib.json             # 存储刀具列表
│   ├── AlphaOptimal.exe         # 软件的可执行文件
│   ├── ...                      # 其他文件
├── plugins/                     # 插件目录
├── translations/                # 翻译文件目录
├── unins000.dat                 # 卸载数据文件
└── unins000.exe                 # 卸载程序
```

> **说明：**
> - 目录结构使用缩进和注释标明每个文件夹和文件的作用。
> - 提供了关键文件的简要说明，便于用户快速理解。
> - 使用了更清晰的格式和对齐方式，提升可读性。
