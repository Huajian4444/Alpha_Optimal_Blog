# 刀具管理

> 刀具列表不为空是所有操作的前提.所以设置刀具是软件操作的第一步,无论需要执行软件的任何功能,请务必先设置必要的刀具

1. 进入刀具管理器
![刀具管理器](../../../image/toolBar.png)

2. 
    1. 已安装
    7

3. 进入刀具管理界面
![刀具管理器界面](../../../image/toolManagerUI.png)
- 添加刀具:新增一把刀具
- 编辑刀具:编辑选定的刀具
- 移动刀具:
    - 把刀具从刀具库移动到已安装,或从已安装移动到刀具库.
    - 这里的移动也包含了数据同步功能
    - 例子：
        ```
            刀具：
            索引：1
            主轴: T1
            刀号: 1
            直径: 20mm
            现在的位置:刀具库
            
            操作:选定刀具->点击:移动刀具

            情况1：已安装 没有索引==1的刀具
                执行结果:复制,会复制一份相同的刀具刀 已安装
            情况2: 已安装 有索引==1的刀具
                执行结果:数据同步.会把 已安装 下,索引==1的刀具的数据，设置为


        ```
- 删除刀具
- 已安装
- 预览刀具
