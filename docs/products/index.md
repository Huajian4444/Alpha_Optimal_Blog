1. 提供了N中功能对加工特征进行调整，无须频繁的修改CAM程序即可实现调整
2. 软件内置算法对轨迹进行了转换和时间优化，和机床动作完全一致。实现了: **所见即所得**
3. 碰撞检测和桥接曲线，这样的一对组合完美的实现了快速检测碰撞并解决碰撞
4. 支持用户夹具的自动&半自动定位

3. 软件整体基于C++开发，决定了它天生就是更快、更高效
